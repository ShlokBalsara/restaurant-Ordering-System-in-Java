# Restaurant Ordering System - Java

🍽️ Simple restaurant ordering system in Java.

## Features
- Menu display 📜
- Order placement 🛒
- Bill generation 💰

## Technologies Used
- Java ☕
- OOP concepts 🏗️

## Setup
1. Clone the repo:
   ```sh
   git clone https://github.com/shlokbalsara/restaurant-ordering.git
   ```
2. Compile and run:
   ```sh
   javac Main.java && java Main
   ```
